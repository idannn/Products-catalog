
1.Cloning HomeAssignment:

	git clone https://github.com/joseph851/Products-catalog.git
	cd HomeAssignment
   
2.Install dependencies:

	npm install


3.Build and serve:

	npm run dev 
	npm run android
	
	if npm run android return error:
	
	npm I jetifier
	npx jetify
	
	and try again:
	
	npm run android
